Pointers

Pointers are variables in programming languages that store the memory location of another variable. Pointers are no different than other variables.The only difference is that a variable stores value of the object and a pointer stores address of some other variable as its value.
For instance,

Let us define a variable x of type int(Integer) and set its value to 7.Each variable contains two informations with it,its value and its memory location in the system.

Now,as we have seen the demonstration of declaration of variable,let us look further at the working of a pointer. Pointer is defined by placing '*' in front of a variable name. '&' character is used as reference to address of variable whose address is to be stored.

But..,How's storing address gonna help me?

  • Passing data to functions
  • String Initialization
  • Dynamic allocation of memory for arrays(malloc)

Passing data to functions

When you pass data to a function using pass by value,system creates a copy of data and pass its value to the parameters(another variable with same value stored at different address) of function.So,when you try to update value of the variable,value of local variable inside function gets updated and the actual value remains unaltered.

Pass By Value

To prevent the duplication of data and this situation,one can pass the required varaible using a pointer.So you are now basically passing the address of the variable whose value can be updated using the dereference operator(*) without duplication of data which further increases the execution speed of the program.
